HTML-perusteet

Sivun rakenne, tagit, CSS ja Bootstrap.

1. Sivun perusrakenne

Jokainen HTML-sivu alkaa samalla rakenteella:

<!DOCTYPE html>
<html lang="fi">
  <head>
    <meta charset="UTF-8">
    <title>Otsikko</title>
  </head>
  <body>
    <h1>Tervetuloa!</h1>
  </body>
</html>
Muista: Elementti = aloitustagi + sisältö + lopetustagi, esim. <p>teksti</p>.
Poikkeus: <br> ja <img> ovat itsesulkeutuvia — ei lopetustageja.

3. CSS-muotoilu

CSS määrittää miltä elementit näyttävät. Rakenne: selektori { ominaisuus: arvo; }

CSS lisätään kolmella tavalla: ulkoisena tiedostona (paras), <style>-lohkona tai inline-tyyleinä (vältä).

/* Elementtiselektori */
h1 { color: blue; text-align: center; }

/* Luokkaselektori */
.korostettu { background-color: yellow; }

/* ID-selektori */
#paaOtsikko { font-size: 2rem; }

Linkitys <head>-osiossa:

<link rel="stylesheet" href="tyyli.css">
4. Bootstrap-kirjasto

Bootstrap antaa valmiit tyylit komponenteille. Lisätään <head>-osioon:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

Bootstrap toimii lisäämällä valmiita luokkia elementeille:

<button class="btn btn-primary">Nappula</button>

<div class="card" style="width: 200px;">
  <div class="card-body">
    <h5 class="card-title">Otsikko</h5>
    <a href="#" class="btn btn-primary">Lue lisää</a>
  </div>
</div>

5. Asettelu — Flex, Grid ja Bootstrap

CSS Flexbox (käytetty tehtävässä 2):

.sailio {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 10px;
}

CSS Grid (käytetty tehtävässä 2 .contact-info:ssa):

.contact-info {
    display: grid;
    grid-template-columns: 200px 200px;
}

Bootstrap Grid — 12-sarakeinen ruudukko:

<div class="row">
  <div class="col-md-6">Vasen puoli</div>
  <div class="col-md-6">Oikea puoli</div>
</div>
Responsiivisuus: Lisää <meta name="viewport" content="width=device-width, initial-scale=1.0"> <head>-osioon — muuten sivu ei skaalaudu oikein puhelimella!
